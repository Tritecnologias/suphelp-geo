🧪 Teste de Busca Híbrida

Este teste valida a funcionalidade de busca híbrida (Local + Google Places API)

📍 Teste 1: Busca em Jundiaí (Dados Locais)

Deve retornar apenas resultados locais, SEM chamar Google Places API

🌐 Teste 2: Busca em São Paulo (Fallback Google)

Deve chamar Google Places API e combinar resultados

🔍 Teste 3: Busca com Filtros

Testa filtros de categoria, rating e telefone

⚠️ Teste 4: Tratamento de Erros

Testa comportamento com coordenadas inválidas

📊 Verificar Logs no Banco

Após executar os testes, verifique os logs no banco de dados:

-- Ver logs de chamadas à API
SELECT * FROM search_cache 
ORDER BY created_at DESC 
LIMIT 10;

-- Ver lugares cacheados do Google
SELECT id, name, address, google_place_id, created_at 
FROM places 
WHERE google_place_id IS NOT NULL 
ORDER BY created_at DESC 
LIMIT 10;
