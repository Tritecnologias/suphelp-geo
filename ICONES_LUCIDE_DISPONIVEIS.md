# Ícones Lucide React Disponíveis

## Como Usar no CMS

Na aba **Recursos** do painel Admin, você pode usar ícones profissionais do Lucide React ao invés de emojis.

### Instruções:
1. Acesse o painel Admin (`/admin`)
2. Vá para a aba **Recursos**
3. No campo **Ícone** de cada card, digite o nome do ícone (ex: `MapPin`, `Download`, `FileText`)
4. Clique em **Salvar Tudo**
5. Os ícones aparecerão automaticamente na landing page

## Notas Importantes

1. **Case Sensitive**: Os nomes dos ícones devem começar com letra maiúscula (ex: `MapPin`, não `mappin`)
2. **Sem Espaços**: Use o nome exato sem espaços (ex: `BarChart3`, não `Bar Chart 3`)
3. **Fallback**: Se o ícone não for encontrado, o sistema usa `MapPin` como padrão
4. **Emojis**: Você ainda pode usar emojis (1-2 caracteres) se preferir
