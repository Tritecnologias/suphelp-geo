# Imagens do SupHelp Geo

## Logo

Salve o logotipo como `logo.png` nesta pasta.

**Especificações:**
- Nome: `logo.png`
- Formato: PNG com fundo transparente
- Tamanho recomendado: 512x512px ou maior
- Fundo: Transparente

**Onde é usado:**
- Header do site (45x45px)
- Páginas de login/cadastro (80x80px)
- Dashboard (50x50px)
- Footer (50x50px)

## Como adicionar:

1. Salve a imagem do logotipo como `logo.png`
2. Coloque nesta pasta: `public/images/logo.png`
3. Faça commit: `git add public/images/logo.png`
4. Push para o servidor

A imagem será redimensionada automaticamente pelo CSS.
